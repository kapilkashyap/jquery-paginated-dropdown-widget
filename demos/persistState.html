Property: persistState (default is false)
This property allows you to persist the state of the drop down, meaning the widget will keep track of the page and even the item that you have navigated to. This will be useful when you have navigated to some item in the drop down but some how the drop down got closed (either by pressing Esc key or by clicking on the close button or by clicking somewhere else on the page, basically blurring the drop down) you can simply go back to the input element and start from your previous state.
#Making the input element blank will reset the widget state.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Setting persistState to true
Sample Code:
$( "#x-demo-2" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@name}</div></div>",
	persistState: true,
	filterJSON: {startsWith:true}
});
