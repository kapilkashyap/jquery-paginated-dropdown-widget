Property: itemTemplate* (default is null)
Setting this property is mandatory. It is used to specify the html construct of individual item in the drop down.
#It is important to know the response of the ajax call in advance.
The response will contain an array of json objects of the following type
{
	"name": "...",
	"email": "...",
	"id": "..."
}
Demo: Setting itemTemplate to "<div title='{@:email}'><div>{@:name}</div><div>{@:email}</div></div>"
Sample Code:
$( "#x-demo-1" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div><div>{@:email}</div></div>",
	filterJson: {startsWith:true}
});
Demo: Setting itemTemplate to "<div title='{@:name}'><div>{@:email}</div></div>"
Sample Code:
$( "#x-demo-2" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:name}'><div>{@:email}</div></div>",
	filterJson: {startsWith:true}
});
