Property: cache (default is false)
This property allows you to set caching for the AJAX calls. Have set this to false by default and leave on the users to decide the value for this property as per their requirement.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith: true}
});
Demo: Setting cache to true
Sample Code:
$( "#x-demo-2" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith: true},
	cache: true
});
