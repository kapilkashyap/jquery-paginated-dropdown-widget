Property: footerRequired (default is true)
This property is used to display the pagination at the bottom of the drop down. If you do not want pagination to be displayed you can set this property to false.
#The footer will be hidden in case the total number of items returned by the ajax call is less than the itemsPerPage property.
#Setting this property to false will also disable next and previous key navigation. Up down key navigation will stay intact until unless specified in the options.
Demo: default behaviour
Sample Code:
$( ".footer-required" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Setting footerRequired to false
Sample Code:
$( ".footer-not-required" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	footerRequired: false,
	filterJSON: {startsWith:true}
});
