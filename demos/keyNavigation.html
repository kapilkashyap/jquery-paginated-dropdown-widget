Property: keyNavigation
This property is used associate keyboard navigation keys with the drop down. Up down, next and previous key navigation is possible on the drop down items. The up_down property under keyNavigation is to toggle the up and down navigation. While next_prev property under keyNavigation is to toggle the ability to navigate through pages using left and right navigation keys.
#However you can navigate through pages using the arrow buttons provided in the footer.
Default values:
keyNavigation: {
	up_down: true,
	next_prev: true
}
Demo: default behaviour
Sample Code:
$( "#key-navigation-0" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Turn off page navigation
Sample Code:
$( "#key-navigation-1" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	keyNavigation: {
		next_prev: false
	},
	filterJSON: {startsWith:true}
});
Demo: Turn off item navigation
Sample Code:
$( "#key-navigation-2" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	keyNavigation: {
		up_down: false
	},
	filterJSON: {startsWith:true}
});
Demo: Turn off keyNavigation
