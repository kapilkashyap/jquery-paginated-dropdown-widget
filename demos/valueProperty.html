Property: valueProperty* (default is null)
Setting this property is mandatory. It is used to specify the property which is going to be used to populate the text input element when the item is selected by either clicking on it or by navigating to the item in the drop down using the keyboard's navigation keys (provided key navigation is not overridden) and pressing enter key.
#It is important to know the response of the ajax call in advance.
The response will contain an array of json objects of the following type
{
	"name": "...",
	"email": "...",
	"id": "..."
}
Demo: Setting valueProperty to name
Sample Code:
$( "#x-demo-1" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJson: {startsWith:true}
});
Demo: Setting valueProperty to email
