Property: elementType (default is text)
This property is used to associate this widget with a link allowing you to click on the link and display the drop down. This can be useful in case you want to show some notifications or alerts.
#The valueProperty is not mandatory in this case.
#The key navigations are disabled by default.
Demo:
Notifications
Sample Html Code:
<a href="javascript:void(0)" class="x-notifications">Notification</a>
Sample jQuery Code:
$( ".x-notifications" ).paginateddropdown({
	url: "assets/notifications.json",
	itemTemplate: "<div title='{@:title}' style='padding: 5px; white-space: normal;'>{@:title}</div>",
	elementType: "link",
	itemsPerPage: 5,
	persistState: false,
	closeBtnRequired: false,
	dropdownWidth: 300,
	customEventPrefix: "lnk"
}).bind( "lnkitemselected", function( event, data ) {
	if(window.console) {
		console.log("callback - lnkitemselected");
		console.log(event);
		console.log(data);
	}
});
