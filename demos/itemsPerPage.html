Property: itemsPerPage (default is 10)
This property is used to determine the max. number of items displayed in the drop down. You can override this property as per your requirement. While overriding this property if the value specified is NaN or less than equal to zero, then the default value will be taken into consideration.
Demo: default behaviour
Sample Code:
$( "#x-demo-1" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	filterJSON: {startsWith:true}
});
Demo: Setting itemsPerPage to 5
Sample Code:
$( "#x-demo-2" ).paginateddropdown({
	url: "assets/cricketers.json",
	valueProperty: "name",
	itemTemplate: "<div title='{@:email}'><div>{@:name}</div></div>",
	itemsPerPage: 5,
	filterJSON: {startsWith:true}
});
